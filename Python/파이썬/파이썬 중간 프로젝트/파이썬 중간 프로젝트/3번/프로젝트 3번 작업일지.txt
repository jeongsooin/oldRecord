4월 15일 오후 4시 48 분
윤년 판별은 함수 호출해서 변수로 받은 담에 변수를 조건문으로 검사하면 될 거 같고
날짜 판별은 조건문을 사용하면 될 거 같다.
1,3,5,7,8,10,12월이 31일 4,6,9,11월이 30 일이니까 
7월 이하에서는 홀수달이 31일, 짝수달이 30일이고
7월 이상에서는 그 반대이다.
이것을 조건문으로 작성한다음에, 7월 이하에서는 2월 달이 있으니까 2월달이 윤년인지 검사하는 조건문을 추가로
작성하면 될 것 같다. 
먼저 원래 작성했던 윤년 판별 함수를 가져오고 메인 함수를 작성한 뒤, GetDayofMonth를 작성해야 겠다.

이 함수는 인풋이 2개인 함수인 것에 주의해야겠다.

코드 작성은 했는데 함수가 2개가 되어서 입력 제어가 안되는듯 오류가 남.
인자이름과 변수 이름은 상관이 없기 때문에 구분을 지어줘야 하는데... 
그럼 get함수에서 호출하지 말고 메인 함수에서 호출하는걸로 바꿔보자.
메인함수에서 입력받은 후 바로 호출하게 했더니 오류 사라짐. 
실행은 되는데 실행 후 바로 꺼진다
실행 후 바로 꺼지지 않도록 코드 추가
